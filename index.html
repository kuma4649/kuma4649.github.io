最新版(latest version)
概要：.vgmや.xgmといった拡張子の付いたファイルを演奏するプログラムです。
bin.zipとbinWithoutVST.zipの違いはVSTが使用できるか否かになります。通常はbin.zipをお使いください。
Summary: This program plays files with extensions such as .vgm and .xgm.
The difference between bin.zip and binWithoutVST.zip is whether or not VSTs can be used. Please use bin.zip in general.
